# Challenge-Alura-Store-Brasil
Esta atividade é um desafio do curso de Data Science 
Este projeto tem como propósito analisar os dados de vendas da Alura Store, comparando o desempenho das lojas, o comportamento dos clientes e os custos envolvidos, com o objetivo de apoiar a tomada de decisão sobre qual loja deve ser vendida. A estrutura do projeto é composta por um notebook principal, chamado AluraStoreBrasil.ipynb, que contém todas as análises, tabelas e gráficos, utilizando dados das quatro lojas carregados a partir de arquivos CSV. Durante a análise, foram gerados diferentes gráficos, como faturamento por loja, vendas por categoria, média de avaliação dos clientes, produtos mais e menos vendidos, frete médio e visualizações geográficas das vendas, permitindo identificar padrões e concluir que uma das lojas apresenta desempenho inferior. Para executar o projeto, é necessário instalar as bibliotecas pandas e matplotlib, abrir o notebook em um ambiente como Jupyter Notebook ou Google Colab e executar as células em sequência.
